Search RNC name

This module searches for the company name by entering a RNC or cedula, if the company is valid.

How does it work?

There are two ways to look up the name with the contact's RNC / Cedula:

  • Enter the RNC or Cedula in the contact's name field, the system will automatically search for the name in the available DGII database. If it cannot find it, the system will send you an error message indicating that it does not exist or is misspelled. If the user is certain that it exists and the system cannot find it, use the second method.
  • Enter the company's name and RNC / Cedula when creating the contact, the system will look up the contact's RNC/Cedula name and will place the correct name. If it cannot find it, it will simply create the contact in a forced manner.

If a contact with a valid tax identification number (RNC) or ID already exists, the system will issue a warning to the user, indicating that they cannot create the contact, as this module restricts the creation of contacts with existing RNCs.
