Pagadito

Pasarela de pago para comercios electrónicos en El Salvador, Guatemala, Honduras, Nicaragua, Costa Rica, Panamá, R. Dominicana, Puerto Rico y USA.

Pagos desde el sitio web de Pagadito.

Ambientes de prueba y producción.

Compatible con Odoo Enterprise y Odoo.sh.

Transacciones

A continuación se detalla los pasos para visualizar las pedidos confirmados desde los pagos realizados en el sitio web.

  1. Ingresar al módulo de Sitio web o website.
  2. Hacer clic en Pedidos dentro del menu pedidos

Configuración

1. Ingresar al módulo de Sitio web o website.

2. Hacer clic en Configuración.

3. En el menu desplegable, hacer clic en métodos de pagos.

4. Entre los diferentes métodos de pagos disponibles, en la nueva ventana, buscar y hacer clic en Pagadito.

5. hacer clic en el botón editar para editar las propiedades del método de pago.

En el campo Estado Seleccione "Modo Test" si desea hacer pruebas. Para producción se debe seleccionar el valor "Habilitado".

En el campo UID, ingrese la clave UID brinda por el sitio pagadito.

En el campo WSK, ingrese la clave WSK brinda por el sitio pagadito.

Pagadito para su funcionamiento, requiere de un diario de pago. Para seleccionarlo, haga clic en la pestaña configuración. En el campo Diario de pago seleccione el diario de pago a usar. También se puede crear uno si así se requiera.

Nota

Si se desea generar una factura apenas una transacción es completada, se deben realizar los siguientes pasos

  1. Desde el módulo sitio web, hacer clic en ajustes dentro del menú configuración.
  2. En la sección Contabilidad, activar la opción Factura automática.

Esta configuración requiere del modulo Contabilidad.

Soporte de Monedas no USD

Se debe tener presente que al momento de realizar pagos con monedas no USD, la plataforma Pagadito hara la conversión de la moneda utilizada en Odoo a USD. A continuación se detallan los diferentes tipos de monedas soportadas:

  • USD Dólares americanos
  • GTQ Quetzales.
  • HNL Lempiras.
  • NIO Córdobas.
  • CRC Colones costarricenses.
  • PAB Balboas.
  • DOP Pesos dominicanos.

Configurando la URL de retorno

Para el correcto funcionamiento de este módulo se requiere configurar la url de retorno cuando el proceso de pago, ha sido completado en la página de pagadito.

1. Para configurar la url de retorno, se requiere acceso al panel de administración de Pagadito.

Dentro del panel principal, buscar la categoría Configuración técnica y hacer clic en Parámetros de integración

En la nueva página, buscar el campo URL. Este campo se encuentro dentro de la sección URL de Retorno.

El valor a ingresar en este campo requiere requiere cumplir el siguiente formato: [URL sitio]/payment/pagadito/return?token={value}&ern={ern_value} donde [URL sitio] es la url completa del sitio. algunos ejemplos a modo de guía: https://www.misitio.com o https://misitio.com o http://misitio.com. Mientras que /payment/pagadito/return?token={value}&ern={ern_value} es un valor estatico y no debe ser alterado.

Tomando como ejemplo la ruta https://www.misitio.com el valor a ingresar en el campo URL sería el siguiente: https://www.misitio.com/payment/pagadito/return?token={value}&ern={ern_value}

Recuerde que https://www.misitio.com es solo un ejemplo, este valor debera cambiarlo por la ruta de su sito web de Odoo.
